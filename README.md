## Stack

```
Next.js 16 · TypeScript · Tailwind CSS v4
shadcn/ui · Framer Motion · Clash Display
```

## Getting Started

```bash
git clone https://github.com/your-username/itrax-lms.git
cd itrax-lms
npm install && npm run dev
```
